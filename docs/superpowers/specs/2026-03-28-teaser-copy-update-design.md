## レイアウト配置

コピーセクションはカウントダウンとメール登録フォームの間に配置する。縦方向の並び順：

1. ロゴ（グロー付き）
2. オープン日
3. カウントダウン
4. **コピーセクション**（タグラインを置換）
5. メール登録
6. Key Facts
7. フッター

## アニメーション

コピーセクションはスタガー（段階的）入場アニメーションを使用：

| 要素 | 初期状態 | アニメーション後 | ディレイ | 再生時間 |
|---------|---------|---------|-------|----------|
| 日本語見出し | `opacity: 0, y: 10` | `opacity: 1, y: 0` | 2.4s | 0.7s |
| 英語見出し | `opacity: 0, y: 10` | `opacity: 1, y: 0` | 2.7s | 0.7s |
| ディバイダー | `opacity: 0, scaleX: 0` | `opacity: 1, scaleX: 1` | 2.9s | 0.5s |
| ボディテキスト | `opacity: 0, y: 10` | `opacity: 1, y: 0` | 3.1s | 0.7s |

後続要素のディレイ調整：

| 要素 | 旧ディレイ | 新ディレイ |
|---------|-----------|-----------|
| メール登録 | 2.7s | 3.5s |
| Key Facts | 3.0s | 3.8s |
| フッター | 3.2s | 4.0s |

## アクセシビリティ

- 日本語見出しは `<h2>` タグでセマンティック階層を表現
- 強調文は `<em>` タグで囲み、CSSでスタイリング
- コントラスト比: `#E6E6E6` on `#000` = 18.1:1、`#8A8A8A` on `#000` = 5.1:1、`#F6FF54` on `#000` = 17.3:1（すべてWCAG AA準拠）
- `prefers-reduced-motion` は既存システムから継承
